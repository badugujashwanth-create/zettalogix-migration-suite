# Changelog

## [0.3.0] - 2026-07-19

### Added

- visibly bounded, network-free synthetic runtime with reset and 13-scene tour;
- evidence-derived migration readiness;
- bundled Lucide icons and local Geist typography;
- Electron renderer capture, desktop build route, CSP, sandbox, and external-link tests;
- expanded web and desktop verification; and
- 5:40 narration, captions, storyboard, and release recorder.

### Changed

- upgraded Vite to 8.1.5, Vitest to 4.1.10, and the React plugin to 6.0.3;
- replaced live-worker claims in demo mode with inspectable runtime boundaries; and
- disabled duplicate start controls and misleading per-job demo report labels.

### Security

- full web and desktop dependency audits report zero vulnerabilities;
- demo mode does not initialize Supabase, Google Picker, or migration API fetches; and
- Electron denies unsafe popup targets.
